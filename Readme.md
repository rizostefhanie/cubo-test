# How to run:

## Run Database container

```  
docker-compose up --build -d 
 ```

## Create .env file with the following variables:

```DB_USER=cubo
DB_PASS=cubo
DB_NAME=admin
DB_HOST=localhost
DB_PORT=27017
```

## Install dependencies
``` 
npm i 
```

## Run project
``` 
npm run serve 

```
